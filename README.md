## Cucumber Framework for a login API (POST) > Receive a token

This project automates the Login functionality of Petstore API Authentication using RestAssured library, Java, and BDD Cucumber. 
Then receive a token
Then use that token in the Authorization header for a second API (GET/POST).
Then validate the response (status code and JSON fields).

## Tech Stack

- Java 23.0.1
- Cucumber BDD
- RestAssured
- Maven
- TestNG
- Cucumber html report
- GitHub version control
---

Run the runner.java file from runner package to initiate execution

## Framework Features

✅ Page Object Model (POM)  
✅ Centralized Configuration  
✅ Tag-based Test Execution (@Smoke, @Regression)  
✅ HTML Reporting  
✅ Jenkins CI/CD friendly
